Java SpringBoot Server (Tomcat) for the Project Dusky (Kakapo redevelopment).

That is a self installing Windows Service. To install, copy this folder whever you want and then open an Administrator
command line window to the new folder. Execute the following commands to install it and start it;

Dusky.exe install
net start Dusky

The following command will print some help documentation. But really you can give it a command of any old garbage
and it will still print the help. See https://github.com/kohsuke/winsw for more detailed information.

Dusky.exe help

Edit application.properties and make any environment specific changes needed for this environment.
